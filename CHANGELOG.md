# Changelog

## [1.0.0] — Initial Release

- Collapsible left tab sidebar with hover-to-expand
- Split terminal views (horizontal and vertical) with drag-to-resize
- Per-directory `.history` files with 60-command dedup, most-recent-first
- History sidebar with one-click re-run and move-to-top on use
- 6 panel header action buttons registered as native VS Code view/title commands
- Themes: VSCode, GitHub Dark, Dracula, Monokai, Solarized Dark, Nord
- Hue / brightness / saturation / opacity sliders
- Alias manager with `.bashrc` / `.zshrc` auto-detection
- Quick Password for fast `sudo` access
- Right-click context menu with package manager shortcuts
- Optional bottom input box
- File drag-and-drop to paste paths
